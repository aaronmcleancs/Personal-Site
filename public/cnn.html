CVV_15M_SARS-CoV-2

COVID-19 X-ray Classification using a 15 Million Parameter Convolutional Neural Network.

Project Overview

This project implements a Convolutional Neural Network (CNN) for classifying chest X-ray images into three categories: COVID-19, Normal, and Viral Pneumonia. It utilizes TensorFlow, AWS, and Keras for training and evaluation on Apple M-series CPUs, achieving high accuracy in diagnosing SARS-CoV-2 from medical images.

Key Features

  • Classifies chest X-ray images into COVID-19, Normal, and Viral Pneumonia categories.
  • Utilizes TensorFlow and Keras for model training and evaluation.
  • Optimized for performance on Apple M-series CPUs.
  • Provides a robust methodology for downloading datasets and model weights.
