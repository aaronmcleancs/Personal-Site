SiteDeck

Web application for analyzing landing page effectiveness using an open vision model.

Project Overview

SiteDeck is a web application designed to automatically analyze sections of websites for areas of improvement in terms of effectiveness, particularly in commerce metrics and visual heatmaps. Utilizing an open vision model, it provides insights to enhance user experience and conversion rates.

Key Features

  • Automated analysis of landing pages for effectiveness.
  • Integration of visual heatmaps to understand user interactions.
  • Insights on commerce metrics to optimize conversion rates.
  • User-friendly interface for quick analysis and reporting.
