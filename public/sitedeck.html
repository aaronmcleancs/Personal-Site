SiteDeck

Web application for analyzing landing page effectiveness using an open vision model.

Project Overview

SiteDeck is an innovative web application that automates the analysis of landing pages to identify areas for improvement in user engagement and conversion optimization. By fetching and rendering target webpages within an iframe, SiteDeck utilizes OpenAI's GPT-4 API to perform comprehensive analyses on various aspects such as legibility, SEO potential, call-to-action effectiveness, and branding consistency. The application presents actionable insights through an interactive interface, aiming to enhance user experience and increase conversion rates. Additionally, SiteDeck dynamically generates visual elements like heatmaps and animated backgrounds based on the extracted color schemes from the webpage, offering a visually engaging way to understand user interactions.

Key Features

  • Automated Analysis Using GPT-4: Leverages OpenAI's GPT-4 API to perform in-depth analysis of webpage content, providing feedback on legibility, SEO, call-to-action elements, and branding.
  • Dynamic Content Rendering: Fetches the target webpage content and displays it within an iframe for real-time analysis.
  • Visual Heatmaps and Dynamic Backgrounds: Generates heatmaps and dynamic background animations based on the webpage's color palette to visualize user engagement areas.
  • Interactive User Interface: Presents analysis results in categorized cards with icons indicating performance levels, enhancing user comprehension.
  • Enhanced User Experience Insights: Provides detailed suggestions to improve user engagement, readability, and overall website effectiveness.

Stack

SiteDeck is built using a combination of modern web technologies:

  • Node.js and Express.js for the backend server to handle API requests and serve content.
  • JavaScript for client-side scripting, including DOM manipulation and asynchronous operations.
  • OpenAI's GPT-4 API for natural language processing and content analysis.
  • HTML5 and CSS3 for structuring and styling the web pages, including responsive design and animations.
  • Heatmap.js and OpenCV.js for generating visual heatmaps and processing images.
  • Fetch API for making network requests to retrieve external webpage content.
  • Font Awesome and Google Fonts for icons and typography.
  • Animate.css for CSS animations.
