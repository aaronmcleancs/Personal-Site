RepBook-DemoServer

Full Stack iOS Fitness Tracking Application with RESTful API support.

Project Overview

RepBook-DemoServer is a backend server for a full-stack iOS fitness tracking application. It provides a personal dashboard for users, allowing them to build workouts, interact with an AI assistant, and manage their fitness metrics through a RESTful API.

Key Features

  • User authentication with signup and login functionality.
  • RESTful API for managing user data and workouts.
  • Integration of an AI assistant for personalized fitness tracking.
  • Ability to create and manage workout plans and exercises.
